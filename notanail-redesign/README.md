## Getting Started

### Installation

```bash
cd notanail-redesign
npm install
```

### Development

```bash
npm run dev
```

The site will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

- **React 18** - UI library
